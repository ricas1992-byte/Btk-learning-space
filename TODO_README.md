# אפליקציית המשימות - TodoList

אפליקציית משימות מודרנית בעברית עם תמיכה מלאה ב-RTL ועיצוב נקי ומינימליסטי.

## ✨ תכונות עיקריות

- **עיצוב מודרני ונקי** - פינות מעוגלות, צללים עדינים וגרדיאנטים
- **תמיכה מלאה ב-RTL** - כל האלמנטים מותאמים לעברית
- **אנימציות חלקות** - כל משימה מופיעה עם אנימציית fade-in
- **אייקונים SVG מינימליסטיים** - במקום אימוג'ים
- **Mobile-first** - מותאם למובייל עם תמיכה ב-safe-area לאייפונים
- **שמירה אוטומטית** - כל השינויים נשמרים ב-localStorage
- **סטטיסטיקות חיות** - מעקב אחרי משימות פעילות והושלמו

## 🎨 עיצוב

- **צבעים**: גרדיאנט זהב/צהוב חם לכפתורים ואלמנטים פעילים
- **טיפוגרפיה**: היררכיה ברורה עם גדלי פונט קריאים
- **צללים**: שימוש עדין ב-box-shadow לעומק ויזואלי
- **אנימציות**: מעברים חלקים של 200-400ms
- **נגישות**: גדלי נגיעה של לפחות 44px, גודל פונט מינימלי 16px

## 📱 התאמה למובייל

- עיצוב responsive מלא
- תמיכה ב-safe-area-inset לאייפונים עם notch
- מניעת זום אוטומטי ב-iOS (font-size: 16px)
- רווחים מספיקים בין אלמנטים
- כפתורים בגודל נוח למגע

## 🚀 שימוש

### דרך 1: דרך האפליקציה הראשית

1. הרץ את הפרויקט: `npm run dev`
2. התחבר לאפליקציה
3. לחץ על "משימות" בתפריט הניווט העליון

### דרך 2: דף עצמאי

1. הרץ את הפרויקט: `npm run dev`
2. פתח את `http://localhost:5173/todo.html`

### דרך 3: שילוב בקומפוננטה אחרת

```jsx
import TodoList from './components/TodoList';

function MyComponent() {
  return (
    <div>
      <TodoList />
    </div>
  );
}
```

## 📂 קבצים

- `src/components/TodoList.jsx` - הקומפוננטה העיקרית
- `src/TodoApp.jsx` - wrapper פשוט להרצה עצמאית
- `todo.html` - דף HTML נפרד
- `src/App.jsx` - אינטגרציה באפליקציה הראשית

## 💾 שמירת נתונים

המשימות נשמרות אוטומטית ב-localStorage תחת המפתח `modern-todo-tasks`.

## 🎯 פונקציונליות

1. **הוספת משימה** - כתוב משימה חדשה ולחץ "הוסף משימה"
2. **סימון כהושלמה** - לחץ על ה-checkbox ליד המשימה
3. **מחיקת משימה** - לחץ על אייקון הפח
4. **מעקב סטטיסטיקות** - ראה את מספר המשימות הכולל, פעילות והושלמו

## 🎨 התאמה אישית

ניתן להתאים את הצבעים על ידי שינוי ה-gradients:
- `from-amber-400 to-yellow-500` - צבע עיקרי
- `from-gray-50 to-gray-100` - רקע

## 📝 הערות טכניות

- הקומפוננטה משתמשת ב-Tailwind CSS
- כוללת אנימציות CSS מותאמות אישית
- תומכת בעד 200 תווים למשימה
- מתאימה לכל גדלי מסך (mobile, tablet, desktop)
